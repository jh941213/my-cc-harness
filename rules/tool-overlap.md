# 도구 오버랩 최소화 정책

## 비활성화된 기본 도구
- **WebSearch**: deny (settings.json에서 차단)
- **WebFetch**: deny (settings.json에서 차단)

## 도구별 역할 분리

| 용도 | 사용할 도구 | 사용 금지 |
|------|-----------|----------|
| 웹 일반 검색 | Tavily MCP | WebSearch |
| 코드 예제/스니펫 검색 | Exa MCP | WebSearch |
| 라이브러리 문서 | Context7 MCP | WebFetch |
| 시맨틱 코드 탐색 | mgrep | — |
| 정확한 문자열/정규식 | built-in Grep, Glob | — |

## 에러 메시지 처리

- 도구가 에러를 반환하면 **절대 무시하지 않는다**
- API 할당량 초과(rate limit) 시: 해당 도구 사용 중단, 사용자에게 알림
- 연속 3회 동일 에러 시: 대체 도구로 전환하거나 사용자에게 에스컬레이션

## 도구 실행 결과 컨텍스트 관리

- 테스트 실행 결과: 성공은 1줄 요약, 실패만 상세 기록
- 긴 로그 출력: 핵심 에러만 추출, 전체를 컨텍스트에 넣지 않음
- 대용량 파일 읽기: 필요한 부분만 offset/limit으로 읽기
