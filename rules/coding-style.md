# 코딩 스타일 (비표준 규칙만)

## 불변성 (필수)
- 항상 새 객체/배열 생성, 절대 뮤테이션 금지
- spread operator, map, filter 등 불변 패턴 사용

## 파일 구성
- 작은 파일 여러 개 > 큰 파일 소수
- 함수: 50줄 이내, 파일: 800줄 이내
- 기능/도메인별로 구성 (타입별 아님)

## 코드 품질
- 깊은 중첩 금지 (4단계 초과 시 추출)
- 이름이 의도를 설명해야 함
- 사용자 입력은 반드시 검증 (Zod 권장)
